# Simple Calculator in C

## Description
This mini project is a menu-driven calculator program written in C.
It performs basic arithmetic operations using switch-case and do-while loop.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with division by zero check)
- User-friendly menu
- Repeats until user exits

## Technologies Used
- Language: C
- Compiler: GCC

## How to Run
1. Compile the program using:
   gcc calculator.c -o calculator
2. Run:
   ./calculator
